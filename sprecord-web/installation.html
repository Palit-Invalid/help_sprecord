Установка на Linux

Поддерживаемые дистрибутивы

  • Основанные на DEB: Ubuntu, Debian, Astra Linux и т.д

 

Процесс установки

Загрузите дистрибутив по ссылке: https://sprecord.ru/files/downloads/sprecord_web_2024_12_05.zip 

На deb дистрибутивах:

wget https://sprecord.ru/files/downloads/sprecord_web_2024_12_05.zip

Разархивируйте архив.

sudo unzip sprecord_web_2024_12_05.zip

Перейдите в директорию.

cd sprecord_web_2024_12_05

Задайте права на запуск.

sudo chmod +x sprecord_web

Запустите программу.

sudo ./sprecord_web

 

После запуска будет доступен WEB интерфейс программы. 

WEB интерфейс доступен в локальной сети если набрать в браузере IP адрес машины на которой запущена программа.

Если у вас дистрибутив Linux с графическим интерфейсом то в браузере наберите http://localhost/
