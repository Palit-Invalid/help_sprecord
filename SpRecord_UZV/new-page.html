Система SpRecord UZV

“Устройство задержки телефонного вызова для аналоговой телефонной линии“

 

Назначение устройства 

Система SpRecord UZV - Устройство задержки вызова (Далее УЗВ) предназначено для выдачи голосового сообщение в телефонную линию перед разговором. УЗВ может использоваться для предварительного уведомления перед началом разговора. УЗВ также позволяет подключить адаптер SpRecord к телефонной линии. 

 

Принцип работы

Перед началом работы на УЗВ записывается голосовой файл. Устройство необходимо подключить по USB к компьютеру. Устройство отобразится как съемный диск. Необходимо записать в корневую папку wav файл в формате 8kHz 16бит моно. По умолчанию там уже записан стандартный файл предупреждения. При входящем вызове УЗВ автоматически «поднимает трубку», не пропуская при этом вызов на телефонный аппарат, затем УЗВ выполняет дозвон на подключенный телефонный аппарат. Удаленный абонент в это время слышит длинные гудки вызова. После поднятия трубки локальный абонент уже не слышит уведомления и может сразу начинать разговор.

Общий вид и описание интерфейсов

 

1. Разъем питания - 5В 2. Разъем для подключения к компьютеру – USB 

Одновременное подключение телефонной линии в разъём - Line и подключение ПК в разъём - USB недопустимо и может привести к повреждению устройства! Перед подключением разъёма USB к ПК необходимо произвести отключение телефонной линии из разъёма Line. 

3. Индикатор статуса устройства - Status 

4. Разъёмы для подключения: Телефонная линия - Line, Телефон - Tel, Адаптер SpRecord – SpRecord

Техника безопасности

Особое внимание уделяйте приведенным ниже инструкциям по технике безопасности: 

- не подключайте одновременно USB кабель и телефонную линию;

- избегайте попадания пыли, грязи и на разъемы смесителя;

- не допускайте механических воздействий, способных повредить изделие;

- не допускайте пролива жидкости на изделие;

- не пытайтесь самостоятельно разбирать изделие. 

Несоблюдение вышеуказанных требований может привести к отказу устройства.
