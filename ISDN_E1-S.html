Замена тонового предупреждения на ГОЛОСОВОЕ для SpRecord ISDN E1-S

Голосовое предупреждение включается по-разному в зависимости от версии установленного программного обеспечения SpRecord. Рекомендуется установить SpRecord версии 3.98 или выше для получения дополнительного функционала адаптеров Е1. Узнать текущую версию SpRecord можно в меню "Справка"-"О программе".

Установка голосового предупреждения для SpRecord версии 3.98 и выше:

  • Выполнить пункт меню Пуск -> "Все программы" -> "SpRecord3" -> "Настроить SpRecord ISDN Terminals Converter". Откроется "Блокнот" с файлом конфигурации.
  • В открывшемся файле конфигурации изменить следующие параметры:

Notification_Incoming = "Voice" 

Notification_Outgoing = "Voice"

 

  • Начиная с этой версии, для входящего и исходящего разговора можно указать разные голосовые фразы: файл "\SpRecord3\ISDN Service\Warning8000.msg" - для входящих, "\ISDN Service\Warning8000Out.msg" - для исходящих разговоров.
  • Сохранить изменения.
  • Перезапустить службу "SpRecord ISDN Terminals Converter".

 

Установка голосового предупреждения для SpRecord версии 3.97 и ниже:
 

  • Зайти под учетной записью с правами администратора на том компьютере, к которому подключено устройство Е1.
  • Остановить службы "SpRecord ISDN Terminals Converter" и "SpRecordService".
  • Удалить "Службу поддержки адаптеров SpRecord ISDN E1" (меню Пуск -> "SpRecord ISDN Terminals Converter" -> "Uninstall SpRecord ISDN Terminals Converter").
  • Скачать по ссылке http://sprecord.ru/download/skachat "Служба поддержки адаптеров SpRecord ISDN E1" файл вида "sritc_setup_xxx.zip" (где xxx - номер версии, начиная с 1.2.х.х и позднее), распаковать и установить.
  • Выполнить от имени администратора пункт меню "Пуск" -> "Все программы" -> "SpRecord ISDN Terminals Converter" -> "Настроить SpRecord ISDN Terminals Converter". 
  • Откроется "Блокнот" с файлом конфигурации.

Изменить следующие параметры:

Включить голосовое предупреждение

Chanel_A_notification_voice = True 

Chanel_B_notification_voice = True

 

  • Сохранить изменения.
  • Перезапустить службы "SpRecord ISDN Terminals Converter" и "SpRecordService". Либо переподключить адаптер.
