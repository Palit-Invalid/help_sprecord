Библиотека распознавания с поддержкой любых цифр в коде региона.

 

Скачать библиотеку https://sprecord.ru/files/downloads/video/region_xxx_libs.zip  

 

Внимание!

Библиотека подходит только для версий IPVR 3.10 и выше.

 

Замена библиотеки под Windows:

1. Остановить службу "IPVideoRecordServer".

2. Заменить файл c:\Program Files (x86)\IPVideoRecord\Server\iANPR_vc14_x86.dll на новый.

3. Запустить службу, проверить работу программы.

 

Замена библиотеки под Linux:

1. Остановить сервис IPVR: "service ipvideoserver stop".

2. Заменить файл /usr/lib/ipvideoserver/libianpr1.82.so на новый.

3. Запустить сервис IPVR: "service ipvideoserver start".

4. Проверить работы программы.
